# Rive MCP Community Store

Free community scripts for [Rive MCP Lite](https://github.com/mysteropodes/rive-mcp-lite).

## How it works

- All scripts in this store are **free to download**
- Creators can add a **support/tip link** (Ko-fi, Buy Me a Coffee, GitHub Sponsors, Patreon, PayPal)
- Some creators sell premium scripts on their own platforms — Lite validates licenses on import

## For creators

Want to publish a script? Use **Rive MCP Launcher (Pro)** to submit your script to this store.

## Structure

```
catalog.json          # Script index (fetched by Lite app)
scripts/
  auto-skin.json      # Individual script files
  create-artboard.json
  ...
```
